# STBM Puskesmas Mobile
STBM Puskesmas Mobile adalah aplikasi berbasis Flutter yang digunakan untuk mendukung program Sanitasi Total Berbasis Masyarakat (STBM) di lingkungan Puskesmas.

Aplikasi ini membantu dalam:
Pendataan kondisi sanitasi masyarakat
Monitoring pilar STBM
